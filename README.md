## How can you contribute?

1. **Fork the Repository**: Click on the 'Fork' button in the upper right corner of the repository page. This will create a personal copy of the repository under your GitHub account.

2. **Clone your Forked Repository**: Open your terminal and run:
   ```bash
   git clone https://github.com/your-username/awesome-for-beginners.git
   ```
   Replace `your-username` with your GitHub username.

3. **Create a Branch**: Navigate to the repository directory and create a new branch for your changes:
   ```bash
   cd awesome-for-beginners
   git checkout -b your-feature-branch
   ```
   Replace `your-feature-branch` with a relevant name for your changes.

4. **Make Changes**: Open the files you want to edit, make your changes and save them.

5. **Commit your Changes**: Stage your changes and commit them with a descriptive message:
   ```bash
   git add .
   git commit -m "Your descriptive message"
   ```

6. **Push your Changes**: Push the changes to your GitHub fork:
   ```bash
   git push origin your-feature-branch
   ```

7. **Open a Pull Request**: Go to the original repository where you want to propose your changes. There will be an option to open a Pull Request from your branch. Click on it, and fill out the necessary details.
